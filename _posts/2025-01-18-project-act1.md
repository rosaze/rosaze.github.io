# Ac.T(관광데이터 공모전) 프로젝트

이 프로젝트는 한국관광공사에서 주최한 관광데이터 공모전 출품작입니다.
프로젝트를 진행하면서 겪은 다양한 기술적 도전과 해결 과정을 카테고리별로 정리했습니다.

## 프로젝트 개요

- 개발 기간: 2024.01 ~ 2024.02
- 사용 기술: React, Node.js, Python
- 팀 구성: 4명

## 주요 기능

1. 관광 데이터 분석
2. 맞춤형 여행 코스 추천
3. 실시간 관광지 정보 제공
